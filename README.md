# Tim Keller 讲道集 / Tim Keller Sermons Collection

提摩太·凯勒 (Tim Keller) 讲道的中英双语整理版本。

A bilingual (English/Chinese) collection of Tim Keller's sermons.

---

## 格式说明 / Format

每篇讲道独立存放于 `sermons/` 子目录中，包含：

Each sermon is stored in its own subdirectory under `sermons/`, containing:

| 文件 / File | 说明 / Description |
|---|---|
| `metadata.json` | 讲道元数据（标题、经文、日期等） / Sermon metadata |
| `outline.md` | 讲道大纲 / Sermon outline |
| `transcript.md` | 英文原文转录 / English transcript |
| `translation.md` | 中文翻译 / Chinese translation |

## 关于 Tim Keller / About Tim Keller

提摩太·凯勒 (Timothy Keller, 1950-2023) 是美国著名牧师、神学家和作家，纽约救赎主长老教会的创始人。他的讲道以深刻的神学洞见和对现代文化的敏锐理解著称。

Timothy Keller (1950-2023) was an American pastor, theologian, and author, founder of Redeemer Presbyterian Church in New York City. His sermons are known for their profound theological insights and keen understanding of modern culture.
